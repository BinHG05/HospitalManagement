# Hospital Management System - Icons

This folder contains icon assets for the application.

## Recommended Icons
- `logo.ico` - Application logo
- `user.png` - User icon
- `doctor.png` - Doctor icon
- `patient.png` - Patient icon
- `admin.png` - Admin icon
- `logout.png` - Logout icon
- `settings.png` - Settings icon
- `appointment.png` - Appointment icon
- `payment.png` - Payment icon

## Icon Guidelines
- Use PNG format for transparency support
- Recommended sizes: 16x16, 24x24, 32x32, 48x48
- Follow consistent color scheme with AppTheme
